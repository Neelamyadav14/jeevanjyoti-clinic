\# Jeevan Jyoti Clinic Management System



A full-stack clinic management system built for Jeevan Jyoti Clinic, Mumbai. Designed to streamline patient appointments, medical records, referrals, and product sales — all in one platform.



\*\*Live Demo:\*\* https://jeevanjyoti-clinic.vercel.app



\---



\## Features



\### Patient

\- Register and login securely

\- Book appointments with symptoms description

\- View appointment history and status

\- Shop for Panchatattva natural skincare products

\- Change password



\### Doctor

\- View and manage all patient appointments

\- Confirm or complete appointments

\- Access full patient history and medical records

\- Add and track referrals to specialists

\- View product orders and revenue

\- Analytics dashboard with charts and breakdowns

\- Change password



\### Shop

\- Browse Panchatattva Natural Soap and Moisturizing Gel

\- Walk-in purchase for small quantities

\- Home delivery available for orders of 10+ units

\- Real-time stock tracking

\## Tech Stack



| Layer | Technology |

|---|---|

| Frontend | React, Vite, Tailwind CSS |

| Backend | Node.js, Express.js |

| Database | PostgreSQL (Supabase) |

| Authentication | JWT (JSON Web Tokens) |

| Deployment | Vercel (Frontend), Render (Backend) |



\---



\## Getting Started Locally



\### Backend Setup

```bash

cd backend

npm install

```



Create a `.env` file in the backend folder:

```

DATABASE\_URL=your\_supabase\_connection\_string

JWT\_SECRET=your\_jwt\_secret

PORT=5000

```



```bash

npm run dev

```



\### Frontend Setup

```bash

cd frontend

npm install

npm run dev

```



Open `http://localhost:5173`



\---



\## Database Schema



\- \*\*users\*\* — patients and doctors with role-based access

\- \*\*appointments\*\* — booking records with status tracking

\- \*\*medical\_records\*\* — diagnosis and prescriptions per patient

\- \*\*referrals\*\* — specialist referral tracking

\- \*\*products\*\* — Panchatattva skincare product catalog

\- \*\*orders\*\* — customer orders with delivery type



\---



\## Deployment



\- Frontend deployed on \*\*Vercel\*\*

\- Backend deployed on \*\*Render\*\*

\- Database hosted on \*\*Supabase\*\*
